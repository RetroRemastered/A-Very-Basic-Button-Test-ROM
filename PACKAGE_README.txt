A Very Basic Button Test ROM
============================

This package contains the ready-to-use Game Boy Advance ROM:

  A Very Basic Button Test ROM.gba

Load the .gba file in a Game Boy Advance emulator, flash cart, or ROM flashing tool.
No build tools are required.

Controls
--------

Press any GBA button. The matching control changes to its pressed artwork while held,
and a square-wave tone plays while the button is held.

Sound mapping
-------------

Left: Do
Down: Re
Up: Me
Right: Fa
Select: So
Start: La
B: Ti
A: Do
L: lowest pitch available from the GBA square-wave channel
R: highest pitch available from the GBA square-wave channel
